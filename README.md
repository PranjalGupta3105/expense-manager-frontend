### Prerequisites

Ensure the following are installed on your system:

- Node.js (v14 or above)
- npm or Yarn
- A running GraphQL backend service

---

### Installation

1. **Clone the repository**

```bash
git clone https://github.com/PranjalGupta3105/expense-manager-frontend.git
cd expense-manager-frontend
```

2. **Install dependencies**

npm install
# or
yarn install


3. **Setup environment variables**

Create a .env file in the project root and add:

VITE_GRAPHQL_ENDPOINT=<your_graphql_api_url>


4. **Run the application**

npm run dev
# or
yarn dev


The app will start on the port shown in the terminal (usually http://localhost:5173).
